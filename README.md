A Sega Genesis-style endless desert racing game built entirely in vanilla HTML/CSS/JS.

Play it: [https://br0thers3bast1an-ops.github.io/dumpster-dash/]

Controls,
| Key | Action |
|-----|--------|
| ↑ / W | Accelerate |
| ↓ / S | Brake |
| ← / → | Steer |
| Z / Shift | Fire missile |

Mobile touch controls are built in.

Features,
Procedurally generated desert highway,
9 difficulty tiers with progressive curve and obstacle scaling,
ETH & BTC coin pickups for missiles,
8-bit synthesized music & sound effects (Web Audio API),
Scrolling mesa/cactus scenery,
Speed-scaled dumpster fire flames
